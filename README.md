# Sonorbit - 基于 Electron 和 Vue 的本地音乐播放器。

## 特点：

- 集现代与复古为一体的 UI ，简约美观
- 匹配网易云歌词（可精确匹配某一首歌）
- 识别内嵌歌词和本地LRC文件
- 丰富且可自定义的快捷键
- 流畅的操作动效
- 旋转封面，呼吸logo
- 桌面歌词



## 未来计划

- 均衡器
- 标签编辑
- 可视化频谱
- 卡拉OK歌词
