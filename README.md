# Sonorbit - 基于 Electron 和 Vue2 的本地音乐播放器。

## 特点

- 集现代与复古为一体的 UI ，简约、流畅、美观
- 匹配网易云歌词（可精确匹配某一首歌）
- 识别内嵌歌词和本地LRC文件，自定义歌词文件夹
- 丰富、可自定义的快捷键
- 旋转封面，呼吸logo
- 桌面歌词，支持纯色和60种渐变色模板
- 歌词模糊效果
- 均衡器
- 可视化频谱
- 标签编辑
