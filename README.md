# Sonorbit - 基于 Electron 和 Vue2 的本地音乐播放器。

## 特点

- 集现代与复古为一体的 UI ，简约美观
- 匹配下载网易云歌词和封面（可精确匹配某一首歌）
- 识别内嵌歌词和本地 lrc 文件，自定义歌词文件夹
- 丰富且可自定义的快捷键
- 流畅的操作动效
- 桌面歌词
- 均衡器
- 标签属性编辑
- 可视化频谱（较耗性能）

## 运行

- npm  install
- npm run serve
- npm start
