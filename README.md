# Loop Sound Player

一个用 Electron 和 Vue 制作的本地音乐播放器。

- 界面美观

- 在线搜索歌词（可精确匹配）

- 识别内嵌歌词和本地LRC文件

- 丰富的全局快捷键

- 流畅的切换动效

- 旋转封面，呼吸logo


## 运行 Electron
